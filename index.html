Privacy Policy for N8N_messenger_automation

Effective Date: January 13, 2026

1. Introduction

N8N_messenger_automation (“the App”) is a Facebook Messenger automation application designed to assist businesses in responding to customer messages through a connected Facebook Page. This Privacy Policy explains how user data is collected, used, stored, and protected.

2. Information We Collect

The App collects and processes the following data:

  • Page-Scoped User ID (PSID)
  • Text message content sent by the user

The App does not collect, store, or use:

  • User names
  • Profile pictures
  • Images, audio messages, videos, or files

3. How We Use Information

  • Maintain conversation context
  • Generate logical and relevant responses
  • Provide automated customer support via Messenger

4. AI Processing

User text messages may be processed by a third-party AI service to generate automated responses. Messages are not used for AI model training and are processed solely for response generation.

5. Data Storage and Retention

  • Data is stored securely using Supabase
  • Data is retained indefinitely to maintain conversation history and response continuity
  • Data is deleted upon user request

6. Data Sharing

User data is not sold, rented, or shared with external organizations. Data is only processed internally for automation and AI response purposes.

9. User Rights

  • Request correction or deletion of data
  • Withdraw consent by discontinuing communication with the Facebook Page
